数据结构概述

  • 定义：我们如何把现实中大量而复杂的问题一特定的数据类型和特定的存储结构保存到主存储器（内存）中，以及在此基础上以实现某个功能（比如查找某个元素，删除某个元素，对所有元素进行排序）而执行的相应操作，这个相应的操作也叫算法。

  • 数据结构=个体+个体关系

  • 算法=对存储数据的操作

  • 算法：1.解题的方法和步骤

    ​ 2.衡量算法的标准：时间复杂度（程序执行次数，而非执行时间）（计算机性能不同）

    ​ 空间复杂度：算法执行过程中大概占用的最大内存

    ​ 难易程度（别人要能看懂）

    ​ 健壮性（正确性就算了，废话，不正确就不叫算法了）

    数据结构的地位：数据结构是软件中最核心的课程

    程序=数据的存储+数据操作+可以被计算机执行的语言

预备知识

指针：指针是C语言的灵魂

定义：地址：内存单元的编号

​ 从0开始的非负整数

​ 范围：0--FFFFFFFF（0-4G-1）

​ 指针：指针就是地址，地址就是指针

​ 指针变量是存放内存单元地址的变量

​ 指针的本质是一个操作受限的非负整数

​ 分类：1.基本类型的指针

​ 2.指针和数组的关系

int i=10;
int *p=&i;//等价于int *p;  p=&i;

#include <stdio.h>
int main(void)
{
    int *p;//P是变量名字，int*表示该p变量只能存储int型变量的地址
    int i=10;
    int j;
    
    p=&i;
    *p=i;//等价于i=i
    printf("i=%d,j=%d,*p=%d\n",i,j,*p);
    return 0;
}

#include <stdio.h>
void f(int i)
{
	i=100;
}
int main(void)
{
	int i=9;//以栈方式压入
	f(i);
	printf("i=%d\n",i);
	return 0
}
